Vinheria Agnello-Sistema de Avaliação de Estoque

Sistema de cadastro e avaliação de estoque de vinhos da Vinheria Agnello, desenvolvido como parte do Checkpoint 02 da disciplina de Web Development.
O sistema permite o cadastro sequencial de múltiplos vinhos via prompt(), realizando validação de entrada, verificação de estoque baixo, classificação por safra e exibição de um relatório final consolidado.
